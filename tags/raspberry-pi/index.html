Raspberry Pi 樹莓派控制門鎖

上一篇的筆記記錄了第一次嘗試使用Raspberry Pi控制電鎖開門，現在記錄一下用Raspberry Pi 控制普通的鎖。普通的門鎖，在門內開鎖是用手去扭或拉一個掣，我們用一條繩的一端綁住鎖上的掣，繩的另一端綁住一個伺服馬達(servomotor)或叫舵機(廣東話不知道是否有另一個名字)，Raspberry Pi 控制舵機轉動把繩往一個方向拉，繼而拉或扭開個掣。
